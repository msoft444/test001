# 要件定義書

## 1. 目的/スコープ
- **目的**: 日付間の日数計算および特定日数後の日付算出を容易に行えるWEBアプリケーションを提供する。
- **スコープ**: ブラウザ上で完結するシングルページアプリケーション（SPA）とし、Dockerコンテナとして配布・実行可能にする。

## 2. 機能要件
- **日数差分計算機能**: ユーザーが「開始日」と「終了日」を指定し、差分（日数）を計算・表示する。
- **日付加減算機能**: ユーザーが「基準日」と「日数（整数）」を入力し、加算/減算を選択して結果日付を表示する。
- **ダークモード切替機能**: トグルスイッチでライト/ダークモードを切り替える。

## 3. 非機能要件
- Dockerコンテナとして構築、環境依存なく起動可能。
- レスポンシブ対応（PC/スマートフォン）。
- 計算処理はクライアントサイドで即座に結果表示。

## 4. 制約・前提
- HTML/CSS/JavaScript + Python 3.11-slim http.server。
- ステートレス（データ永続化なし）。
